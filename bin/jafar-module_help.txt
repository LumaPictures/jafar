Usage: jafar-module [OPTIONS] [MODULE] [EXTRA FILES]

Jafar tool to create a new module or update an existing module.

Options:

  -h, -help, --help: display this help message

  -c, --create: create a directory MODULE in JAFAR_DIR/modules
  including files to start a new jafar module.

Examples:

  jafar-module -c helloworld
